# Pick-and-Place-Lego-Robot
Pick and place a ball using Lego Mindstorm EV3 robotic arm

# Introduction 
Manipulators, versatile systems designed for precise control and execution of tasks, consist of three rotary actuating motors: Motor A, Motor B, and Motor C. Complementing these motors are five touch sensors, with Touch Sensor #1 and Touch Sensor #2 providing essential feedback, and three encoders (Motor A encoder, Motor B encoder, and Motor C encoder) ensuring accurate control. This integrated network of motors and sensors equips the manipulator for intricate pick and-place operations, showcasing adaptability and precision in various tasks.
